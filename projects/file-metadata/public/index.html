 API Documentation

        
        

          

            POST
            /api/fileanalyse
          

          

            Upload a file using multipart/form-data. The file input field must have the name upfile.
          

          

            
Example Response:

            
{
  "name": "example.png",
  "type": "image/png",
  "size": 12345
}

          

        


        
        

          

            HTML
            Form Example
          

          

            
HTML Form:

            
<form action="/api/fileanalyse" method="POST" enctype="multipart/form-data">
  <input type="file" name="upfile" />
  <button type="submit">Upload</button>
</form>
