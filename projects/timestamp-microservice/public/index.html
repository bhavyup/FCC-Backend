Valid Date String

Unix Timestamp

  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Current Time

GET /api 
{
  "unix": [current timestamp],
  "utc": "[current UTC time]"
}

Invalid Date

GET /api/invalid-date 
{
  "error": "Invalid Date"
