              POST
              /api/shorturl
            

            
Send a url field (form or JSON). Returns original_url and numeric short_url.

            
curl -X POST -d "url=https://example.com" \
  localhost:3002/api/shorturl

          


          

            

              GET
              /api/shorturl/:id
            

            
Redirects (302) to the original URL. Returns {error} if not found.

            
curl -L localhost:3002/api/shorturl/1

          


          

            

              GET
              /api/urls
            

            
Returns all shortened URLs with total count.

          


          

            
Validation

            
Must use http:// or https:// and resolve via DNS.

            

              https://example.com ✓
              http://example.com/path ✓
              ftp://files.com ✕
              www.example.com ✕
