                    API Documentation
                


                

                    
                    

                        

                            POST
                            /api/shorturl
                        

                        

                            Submit a URL to be shortened. Send the URL in the request body.
                        

                        

                            
Request Body:

                            
{ "url": "https://www.google.com" }

                        

                        

                            
Success Response:

                            
{
  "original_url": "https://www.google.com",
  "short_url": 1
}

                        

                        

                            
Error Response:

                            
{ "error": "invalid url" }

                        

                    


                    
                    

                        

                            GET
                            /api/shorturl/:id
                        

                        

                            Visit a shortened URL to be redirected to the original URL.
                        

                        

                            
Example:

                            
GET /api/shorturl/1
