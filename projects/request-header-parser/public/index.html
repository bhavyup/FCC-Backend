API Documentation

GET /api/whoami

Returns a JSON object containing your IP address, preferred language, and software (User-Agent) information extracted from request headers.

Example Response:
{
  "ipaddress": "159.20.14.100",
  "language": "en-US,en;q=0.9",
  "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)..."
}

Understanding HTTP Headers

X-Forwarded-For / IP

Your public IP address, identifying your network location on the internet.

Accept-Language

Languages your browser prefers, helping servers deliver localized content.

User-Agent

Information about your browser, operating system, and device.

Try with cURL

curl [your-url]/api/whoami
